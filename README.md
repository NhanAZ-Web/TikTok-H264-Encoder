# TikTok H.264 Encoder

Web app that re-encodes video to H.264 / AAC using FFmpeg WASM in the browser. Select or drag-and-drop a video file, wait for processing, then download.

## Run

Requires Node.js. From the project folder:

```bash
node server.js
```

Open your browser at `http://localhost:8765`.
